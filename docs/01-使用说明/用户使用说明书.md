## 1. 产品介绍

### 1.1 产品概述

**社区洞察全景 Akashic** 是一个基于12D框架的社区产品分析系统，整合人类学、心理学、政治学、经济学、网络科学等多学科理论，提供200+实战案例库和AI智能分析工具。

### 1.2 核心价值

- **多学科理论整合**：12个理论维度，深度解析社区产品底层逻辑
- **实战案例库**：200+社区产品案例，覆盖各种类型和阶段
- **AI智能分析**：基于Gemini AI的智能诊断和对比分析
- **实用工具箱**：产品体检、游戏化架构、增长模拟等专业工具

### 1.3 适用人群

- 社区产品经理
- 产品设计师
- 运营人员
- 创业者
- 研究者
- 投资人

## 3. 核心功能

### 3.1 12D理论框架

系统提供12个理论维度，帮助用户从不同角度理解社区产品：

1. **人类与心理**：进化心理学视角，理解用户行为驱动
2. **空间与交往**：空间设计如何塑造用户行为
3. **政治与治理**：社区治理模式（君主制、联邦制、算法统治）
4. **经济与价值**：价值创造与分配机制
5. **社交与连接**：强关系、弱关系、潜在关系的价值
6. **生态与演化**：社区生态系统的演化规律
7. **宗教与仪式**：仪式如何创造身份认同
8. **叙事与旅程**：英雄之旅与集体叙事
9. **行为与决策**：认知偏差与决策设计
10. **游戏与虚拟**：游戏化运营与激励机制
11. **结构与流动**：网络结构与信息流动
12. **趋势与变革**：行业趋势与未来方向

### 3.2 案例库（200+案例）

**案例类型：**
- 通用社交（微信、抖音、快手等）
- 垂直社区（B站、小红书、知乎等）
- 游戏社区（Discord、Steam等）
- 工具社区（GitHub、Stack Overflow等）
- 内容社区（YouTube、B站等）
- 其他类型

**案例信息：**
- 5D基因指纹（雷达图）
- 详细维度分析
- 时间线演进
- 关键决策点
- 失败教训（失败案例）

### 3.3 竞品对比实验室

**功能特性：**
- 自由对战模式（2产品对比）
- 商业混战模式（3-5产品对比）
- 多维度对比分析
- 红蓝PK可视化
- 差异数据展示

### 3.4 工具箱

**可用工具：**
1. **产品体检仪**：输入参数，获得5D模型诊断
2. **游戏化架构师**：生成PBL激励体系
3. **社区画布生成器**：生成产品定义文档
4. **竞品对比实验室**：产品对比分析
5. **生态演化模拟器**：预测社区生态变化
6. **增长模拟器**：用户增长趋势预测
7. **其他工具**：优先级矩阵、用户画像生成器等

## 4. 功能详解

### 4.1 案例库使用

#### 4.1.1 浏览案例

1. 点击顶部导航栏的 **"13. 案例与竞品"**
2. 在案例库中浏览所有案例卡片
3. 点击任意案例卡片进入详情页

#### 4.1.2 筛选案例

- **按类型筛选**：点击类型按钮（全部、通用社交、垂直社区等）
- **按标签筛选**：点击标签按钮（#强关系、#算法等）
- **搜索筛选**：在搜索框输入关键词
- **排序**：选择排序方式（按名称、按5D评分等）

#### 4.1.3 案例详情页

**页面内容：**
- 产品基本信息（图标、标题、描述）
- 5D基因指纹雷达图
- 多维度分析（13个模型）
- 时间线演进
- 关键决策点
- 用户画像
- 商业模式
- 关键指标图表

**操作功能：**
- 收藏案例（❤️ 按钮）
- 评分案例（⭐ 按钮）
- 添加笔记
- AI深度诊断
- 查看相似案例

### 4.2 竞品对比使用

#### 4.2.1 自由对战模式（2产品对比）

1. 点击 **"竞品对比实验室"**
2. 选择挑战者A和挑战者B
3. 选择分析模型（5D、生命周期、增长模型等）
4. 查看对比结果：
   - 雷达图对比
   - 数据卡片对比
   - 差异分析
   - 图表对比

#### 4.2.2 商业混战模式（3-5产品对比）

1. 点击 **"商业混战模式"**
2. 选择3-5个产品
3. 选择分析模型
4. 查看多产品对比：
   - 多产品数据卡片
   - 多产品图表对比
   - 数据表格对比

### 4.3 工具箱使用

#### 4.3.1 产品体检仪

1. 进入工具箱，点击 **"产品体检仪"**
2. 输入产品参数：
   - 社交维度（0-100）
   - 公共维度（0-100）
   - 治理维度（0-100）
   - 算法维度（0-100）
   - 经济维度（0-100）
3. 点击 **"开始诊断"**
4. 查看诊断报告：
   - 5D雷达图
   - 维度分析
   - AI诊断建议

#### 4.3.2 游戏化架构师

1. 点击 **"游戏化架构师"**
2. 输入产品信息
3. 生成PBL体系（积分、徽章、排行榜）
4. 导出方案

#### 4.3.3 增长模拟器

1. 点击 **"增长模拟器"**
2. 设置参数：
   - 初始用户数
   - 增长率
   - 病毒系数
   - 时间周期
3. 查看增长预测图表
4. 导出报告

## 5. 使用技巧

### 5.1 快捷键

- `/` 键：快速聚焦搜索框
- `ESC` 键：关闭搜索框或弹窗
- `1-8` 数字键：快速切换视图（部分视图）

### 5.2 数据持久化

- **自动保存**：所有操作（收藏、评分、笔记）自动保存到浏览器本地存储
- **刷新保留**：刷新页面后，收藏、评分、笔记等数据会保留
- **清除数据**：清除浏览器缓存会清除所有本地数据

### 5.3 导出功能

**支持导出格式：**
- PDF报告
- Excel表格
- CSV数据
- 图片（截图）

**导出位置：**
- 案例详情页：导出案例报告
- 竞品对比页：导出对比报告
- 工具箱：导出分析结果

### 5.4 最佳实践

1. **案例学习**：
   - 先浏览案例库，找到感兴趣的类型
   - 查看案例详情，理解5D基因指纹
   - 对比相似案例，发现差异和规律

2. **竞品分析**：
   - 选择2个直接竞品进行对比
   - 使用多个模型分析，全面理解差异
   - 导出对比报告，用于产品决策

3. **产品设计**：
   - 使用产品体检仪评估现有产品
   - 参考相似案例的设计模式
   - 使用工具箱生成设计方案

## 6. 常见问题

### 6.1 性能问题

**Q: 页面加载较慢？**  
A: 首次加载需要加载大量数据，请耐心等待。建议使用现代浏览器（Chrome、Firefox、Edge）。

**Q: 图表不显示？**  
A: 请检查浏览器控制台是否有错误，确保网络连接正常。图表库需要从CDN加载。

### 6.2 功能问题

**Q: 搜索功能不工作？**  
A: 请确保搜索关键词正确，搜索支持中文和英文。

**Q: AI诊断失败？**  
A: AI功能需要网络连接，请检查网络状态。如果持续失败，可能是API限制。

**Q: 数据丢失？**  
A: 数据保存在浏览器本地存储中，清除浏览器缓存会导致数据丢失。建议定期导出重要数据。

### 6.3 兼容性问题

**Q: 哪些浏览器支持？**  
A: 支持现代浏览器：Chrome 90+、Firefox 88+、Edge 90+、Safari 14+。

**Q: 移动端可以使用吗？**  
A: 可以，但建议使用桌面浏览器获得最佳体验。
