### 2. 新增图表类型

#### 2.1 用户流转漏斗图 (userFlowFunnelChart)
- **类型**: 水平条状图（bar chart with indexAxis: 'y'）
- **用途**: 展示用户从访问到推荐的转化流程
- **特点**:
  - 显示6个阶段：访问、注册、激活、留存、付费、推荐
  - 渐变色方案（蓝色到粉色）
  - 工具提示显示转化率
- **函数**: `renderUserFlowFunnel(canvasId, data)`
- **状态**: ✅ 已完成

#### 2.2 价值流动图 (valueFlowChart)
- **类型**: 多系列折线图（line chart）
- **用途**: 展示价值流动和成本流动的对比
- **特点**:
  - 双系列：价值流量（绿色）和成本流量（红色）
  - 6个节点：内容创作、分发、消费、广告、订阅、电商
  - 填充区域显示趋势
- **函数**: `renderValueFlowChart(canvasId, data)`
- **状态**: ✅ 已完成

#### 2.3 理论关联矩阵 (theoryCorrelationMatrix)
- **类型**: 水平条状图（bar chart with indexAxis: 'y'）
- **用途**: 展示11个理论页面之间的关联度
- **特点**:
  - 基于智慧胶囊标签的Jaccard相似度计算
  - 颜色渐变（绿色=高关联，红色=低关联）
  - 显示每个理论与其他理论的最大关联度
- **函数**: 
  - `generateTheoryCorrelation()` - 生成关联数据
  - `renderTheoryCorrelationMatrix(canvasId)` - 渲染图表
- **状态**: ✅ 已完成

## 📝 技术实现细节

### 配色提取逻辑
理论卡的header背景色通过 `renderWisdomCard` 函数从 `theme.pattern.border` 和 `theme.insight.border` 中提取：
- Pattern类型：使用主色（如 `border-blue-500` → `bg-blue-500`）
- Insight类型：使用辅助色（如 `border-cyan-500` → `bg-cyan-500`）

### 理论关联度计算
1. **标签提取**: 从每个理论页面的智慧胶囊中提取所有标签
2. **Jaccard相似度**: 计算两个理论页面标签的交集/并集
3. **最大关联度**: 每个理论与其他理论的最大相似度
4. **颜色映射**: 关联度越高，颜色越绿；关联度越低，颜色越红

### 图表实现方式
- **漏斗图**: 使用水平条状图模拟漏斗效果
- **价值流动图**: 使用双系列折线图展示价值流和成本流
- **关联矩阵**: 使用水平条状图展示理论关联度排名
