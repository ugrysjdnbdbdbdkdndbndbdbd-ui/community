- **第三背景（BG Tertiary）**：`#1f2937` - 中灰色

- **边框（Border）**：`rgba(0, 212, 255, 0.2)` - 半透明科技蓝

## ✨ 视觉特效

### 发光效果（Glow）

1. **主发光（Glow Primary）**
   ```css
   box-shadow: 0 0 20px rgba(0, 212, 255, 0.3);
   ```

2. **强发光（Glow Primary Strong）**
   ```css
   box-shadow: 0 0 30px rgba(0, 212, 255, 0.5);
   ```

3. **强调发光（Glow Accent）**
   ```css
   box-shadow: 0 0 15px rgba(0, 245, 255, 0.4);
   ```

### 阴影系统（带发光）

- **小阴影（Shadow SM）**：基础阴影 + 弱发光
- **中阴影（Shadow MD）**：多层阴影 + 中发光
- **大阴影（Shadow LG）**：多层阴影 + 强发光
- **超大阴影（Shadow XL）**：多层阴影 + 超强发光
- **发光阴影（Shadow Glow）**：纯发光效果

### 渐变效果

1. **按钮渐变**
   ```css
   background: linear-gradient(135deg, #00d4ff 0%, #06b6d4 100%);
   ```

2. **背景渐变**
   ```css
   background: linear-gradient(135deg, #0a0a0a 0%, #111827 50%, #0a0a0a 100%);
   ```

3. **边框渐变**
   ```css
   border: 1px solid;
   border-image: linear-gradient(90deg, transparent, #00d4ff, transparent) 1;
   ```

## 🎬 动画效果

### 1. 脉冲发光（Pulse Glow）
```css
@keyframes pulse-glow {
    0%, 100% { opacity: 0.5; }
    50% { opacity: 1; }
}
```

### 2. 渐变动画（Gradient Animation）
```css
@keyframes animate-gradient {
    0%, 100% { background-position: 0% 50%; }
    50% { background-position: 100% 50%; }
}
```

### 3. 光扫效果（Shimmer）
```css
.btn-primary::before {
    background: linear-gradient(90deg, transparent, rgba(255, 255, 255, 0.3), transparent);
    animation: shimmer 0.5s;
}
```

### 4. 淡入上移（Fade In Up）
```css
@keyframes fadeInUp {
    from {
        opacity: 0;
        transform: translateY(20px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}
```

### 5. 旋转（Rotate）
```css
@keyframes rotate {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
}
```

使用CSS自定义属性（CSS Variables）实现主题系统，便于后续维护和扩展。

### 过渡动画
- **快速（Fast）**：150ms
- **基础（Base）**：300ms
- **慢速（Slow）**：500ms

## 🎯 设计理念

### 科技感
- 深色背景营造专业感
- 科技蓝/青色作为主要强调色
- 发光效果增强未来感

### 可读性
- 高对比度确保文字清晰
- 层次分明的文字颜色系统
- 适当的间距和留白

### 交互性
- 丰富的悬停效果
- 平滑的过渡动画
- 清晰的视觉反馈

### 一致性
- 统一的配色方案
- 一致的组件样式
- 协调的动画效果

## 📝 使用说明

### 自定义颜色
如需调整主题颜色，只需修改CSS变量：
```css
:root {
    --color-primary: #00d4ff;  /* 修改主色 */
    --color-bg-card: #1a1a2e;  /* 修改卡片背景 */
    /* ... */
}
```

### 添加新组件
新组件应遵循以下原则：
1. 使用CSS变量而非硬编码颜色
2. 添加适当的发光效果
3. 实现平滑的过渡动画
4. 确保在深色背景下的可读性
